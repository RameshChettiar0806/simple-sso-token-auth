Simple Single Sign-On (SSO) System

This project demonstrates a backend-focused authentication system using Email + OTP, JWT, and RSA-based digital signatures to enable secure Single Sign-On across services.

How Authentication Works

  1. Enter your email address.
  2. Verify using a one-time password (OTP).
  3. Receive a signed JWT for SSO access.

Login





No password is required. Authentication is completed using a secure OTP.
